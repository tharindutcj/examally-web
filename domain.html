Literature survey

  • Automatic Question Generation System from Paragraph was a research done to present an approach to generate questions from a paragraph and the size of the paragraph is defined by its scope.

  • Automatic Question and Answer Generation from Course Materials was the system based on the approach of Natural Language Processing to analyze the text sentence by analyze the text sentence by sentence and generate all possible questions.

  • Automatic Question Generation System Using Software Agents for Technical Institutions was developed and implemented was to take input in form of a text file from user which contains of the text upon which the user desires to fetch questions.

  • Automatic Question Generation from Text, extracted and simplified multiple statements from complex sentences including operations for various syntactic constructions encoded with pattern matching rules for trees.

Research gap

  • Existing systems take only textual documents as the input.

    • In the proposed system, audio files will also be taken as an input to overcome this limitation.

  • Existing systems have failed to provide a wide range of functionalities, rather narrowed down to a specific topic.

    • Proposed system will include both the automatic question generation and lesson summarization functionalities.

Research Problem

  • Difficulty in creating question based on syllabus.

  • Doing related questions and referring to a summary of the lesson make the process much more effective, efficient, interesting and easier.

  • But it takes time,

    • To summarize a lesson.

    • Search and find related questions.

Research objectives

  • Main intention in this research is to, come up with a solution by implementing an intelligent questions and answer generating and also a summarizing system for teachers and students via user-friendly web application.

  • Provide better procedure to automatically generate various kind of questions and a summary to get a quick understand about the lesson.

  • To save the precious time and money by providing important and validated questions and answers with high accuracy.

Methodology

  • Audio Analyzation

    • Separating the main audio source and enhance it.

    • Converting the audio to text.

    • Filtering unnecessary information and abstract only what is related to the lesson topic.

  • Text Extraction and Question Generation

    • Extracting the text from the uploaded document.

    • Identifying and filtering the content where questions can be generated.

    • Generating “Wh”, yes/no and gap-fill questions from the identified content and provide answers to them.

  • MCQ Generation

    • Identifying content where MCQs can be generated and generating the MCQ.

    • Generating invalid choices for the questions which are similar to the correct choice.

    • Filtering similar questions.

  • Question Validation and Summary Generation

    • Analyzing and removing similar questions and questions with grammar errors.

    • Identifying key points in the text.

    • Generating a summary from the identified key points.
